# Repositorio de Soluciones

Es un repositorio donde almaceno las soluciones aplicando TDD de los retos, los pruebas de certificaciones o katas que hago de plataformas como HackerRank o problemas semanales que lanzan algunos creadores de contenido
como Mouredev o Carlos Azaustre.

## Descargo de responsabilidad

No utlizar el código literalmente en los retos o para obtener los certificados. Empleelos como inspiración
o como un propio ejercicio de análisis al enfretarse a estos retos.
